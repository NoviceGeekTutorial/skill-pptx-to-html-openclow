# skill-pptx-to-html-openclaw - 兼容 OpenClaw 架构的 PPT 转 HTML 专业工具

将 PowerPoint 演示文稿转换为高精度 HTML，**1:1 完整复刻**原版布局、颜色和排版，生成文字可选中复制的网页，**完美兼容 OpenClaw 架构**。

## 核心特性
- **1:1 布局复刻**：精确还原所有元素的位置、尺寸和样式
- **文字可选中复制**：HTML 中的所有文本都可以自由选择和复制
- **颜色精确还原**：从形状、文本、背景中精确提取颜色值
- **图片自动提取**：自动提取并优化嵌入的图片
- **响应式设计**：自适应桌面、平板、手机等多种设备
- **跨平台支持**：支持 Windows、macOS、Linux
- **OpenClaw 架构兼容**：可在 OpenClaw 环境中稳定运行、集成与部署

## 支持的元素类型
| 元素类型 | 支持情况 |
|---------|---------|
| 文本框 | ✅ 完整支持（包含字体、颜色、对齐） |
| 图片 | ✅ 完整支持（自动提取） |
| 自定义形状 | ✅ 完整支持（矩形、圆形、箭头等） |
| 表格 | ✅ 完整支持（包含样式和颜色） |
| 图表 | ⚠️ 视觉渲染（导出为图片） |
| 占位符 | ✅ 完整支持 |
| 组合形状 | ✅ 完整支持 |
| 幻灯片背景 | ✅ 颜色提取 |
| 渐变 | ⚠️ 简化处理为纯色 |
| 动画 | ❌ 不支持（静态输出） |

## 使用方法
### 基本转换
```bash
python scripts/pptx_to_html.py <输入文件.pptx> [输出目录]
 
 
示例
 
bash
  
# 转换演示文稿
python scripts/pptx_to_html.py "我的演示文稿.pptx" ./输出目录

# 输出文件：
# ./输出目录/我的演示文稿.html
# ./输出目录/assets/pptx-styles.css
# ./输出目录/assets/images/
 
 
输出文件结构
 
plaintext
  
输出目录/
├── [文件名].html              # 主 HTML 文件
└── assets/
    ├── pptx-styles.css       # 样式表
    └── images/               # 提取的图片
        ├── slide_1_img_1.png
        ├── slide_1_img_2.jpg
        └── ...
 
 
工作流程
 
1. 分析 PPTX 文件
 
使用  python-pptx  库提取：
 
- 幻灯片尺寸和背景颜色
- 形状的位置、大小和层级顺序
- 文本内容及其格式（字体、大小、颜色、加粗、斜体）
- 图片及其属性
- 表格结构和样式
 
2. 生成 HTML
 
- 使用绝对定位确保精确布局
- 内联样式保持颜色和排版
- CSS 类实现响应式行为
- 图片使用相对路径引用
 
3. 创建 CSS
 
- 响应式断点适配不同屏幕尺寸
- 打印友好的样式
- 移动端优化的布局
 
依赖安装
 
bash
  
pip install python-pptx Pillow
 
 
技术细节
 
坐标系统
 
- PPTX 使用 EMU（英制公制单位）：1 英寸 = 914,400 EMU
- HTML 使用像素：默认按 96 DPI 转换
- 所有位置在幻灯片容器内使用绝对定位
 
文本处理
 
- 字体大小以磅（pt）为单位保留
- 颜色提取为 RGB 十六进制值
- 文本对齐映射到 CSS text-align
- 字体名称保留并添加回退字体
 
图片提取
 
- 原始图片格式保留（PNG、JPEG 等）
- 图片保存到  assets/images/ 
- HTML 中的引用路径已更新
 
响应式行为
 
在较小屏幕上：
 
- 幻灯片按比例缩放
- 文本保持可读
- 布局比例保持不变
 
OpenClaw 兼容说明
 
- 支持在 OpenClaw 架构环境中直接调用与运行
- 无额外依赖，可无缝集成至 OpenClaw 生态工具链
- 保持轻量化、高效率、跨平台一致性
 
已知限制
 
1. 动画：PowerPoint 动画不会转换
2. 切换效果：幻灯片切换效果不会保留
3. 音视频：嵌入式媒体不会提取
4. 复杂渐变：可能简化为纯色
5. 特效：阴影、光晕、3D 效果可能无法精确渲染
 
故障排除
 
文本不显示
 
- 检查文本是否在组合形状中
- 确认系统上是否有所需字体
 
图片缺失
 
- 检查  assets/images/  目录是否存在
- 验证图片提取权限
 
布局问题
 
- 确保以 100% 缩放查看输出
- 建议使用 Chrome/Firefox/Edge 浏览器

- ✅ 形状填充颜色
- ✅ 形状边框颜色
